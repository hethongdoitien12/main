# Phase 12 — Activity Feed & Social Activity System

Đọc replit.md và tientrinhethong.md trước.

Mục tiêu:

Biến MT Economy thành nền tảng "sống".

Người dùng luôn thấy hoạt động mới.

---

## Database

activity_feed

Fields:

- id
- actor_id
- actor_username
- actor_avatar
- activity_type
- target_id
- target_name
- amount_mt
- metadata
- created_at

Indexes:
- created_at DESC
- activity_type

---

## Activity Types

TIP_SENT

Ví dụ:

user_demo_01 tipped creator_nam 500 MT

---

PRODUCT_PURCHASED

Ví dụ:

user_demo_03 purchased React Dashboard Template

---

FANCLUB_JOINED

Ví dụ:

user_demo_05 joined creator_huy Gold Fan Club

---

CREATOR_VERIFIED

Ví dụ:

creator_linh became a Verified Creator

---

CREATOR_FEATURED

Ví dụ:

creator_khoa is now Featured

---

MILESTONE_REACHED

Ví dụ:

creator_huy reached 10,000 MT earned

---

## Backend

POST activity automatically when:

- Tip success
- Product purchase
- Fan Club join
- Verification
- Featured
- Milestone

API:

GET /api/activity

Query:

- limit
- page
- activity_type

Return latest first.
